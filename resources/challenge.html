Checking if you're a bot

The process is entirely automatic. If you encounter issues, contact botnet@zajc.tel.

This page expends some computational power to solve a generated challenge. This does not benefit anyone, but it does keep unwanted bots out - the process is fast for individual users, but very slow in bulk. Your IP address is whitelisted once the process concludes to avoid additional interruptions.

JavaScript is unfortunately required for this. If you'd like, you can view the source here: challenge.js, challenge-worker.js, challenge-settings.js (generated). You can also view the full source of the project

The progress bar is only an estimate. If it gets stuck, wait longer or try reloading.
